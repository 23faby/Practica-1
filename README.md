Shadowsocks for Windows
=======================

#### Features compare with official version

1. Add a plugins interface
2. Add a PingPlugin for Demo

#### 和官方版本相比有什么不同

1. 新增插件功能
2. 增加了Ping测试的Demo插件，如果需要在ping过程中获取获取服务器IP归属地，请下载[纯真IP库]并把qqwry.dat文件放置在当前目录
